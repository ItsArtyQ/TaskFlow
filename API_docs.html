Overview

TaskFlow API — REST API для системы управления задачами. API построен по REST-принципам и использует JWT авторизацию.

Base URL

http://localhost:3000/api

Content-Type

application/json

Authentication

POST /auth/register

Request Body

Field Type Description
email string User email
username string Username
password string User password 
{
  "email": "user@example.com",
  "username": "artem",
  "password": "12345678"
}

Response

{
  "data": {
    "user": {
      "id": "uuid",
      "email": "user@example.com",
      "username": "artem"
    },
    "accessToken": "jwt-token"
  }
}
POST /auth/login

Users

GET /users/me

Headers

Authorization: Bearer jwt-token

Response

{
  "data": {
    "id": "uuid",
    "email": "user@example.com",
    "username": "artem",
    "createdAt": "2026-05-17T12:00:00.000Z"
  }
}

Boards

POST /boards

Request Body

{
  "title": "Backend Development"
}
GET /boards
PATCH /boards/:id
DELETE /boards/:id

Tasks

POST /tasks

Request Body

{
  "title": "Implement JWT",
  "description": "Create access token flow",
  "boardId": "uuid"
}
GET /tasks
PATCH /tasks/:id
DELETE /tasks/:id

Comments

POST /tasks/:id/comments

Request Body

{
  "content": "Need to improve validation"
}

Error Format

{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Invalid email"
  }
}

HTTP Statuses

200 OK Successful request
201 Created Resource created
400 Bad Request Validation failed
401 Unauthorized Invalid token
403 Forbidden No permissions
404 Not Found Resource not found
